SmartArt Component - Embed Examples

Standard Embed
This is the standard way to embed the SmartArt component. It includes the full interactive experience with AI-powered features.
How to Use This Code
Copy and paste this HTML code into your website where you want the SmartArt component to appear:
<iframe src="https://shens2.github.io/smartart/" width="800" height="500" frameborder="0" allowtransparency="true" style="border: 1px solid #e0e0e0; border-radius: 8px;"> </iframe>
Responsive Version
For responsive websites, use this code which adapts to different screen sizes:
<div style="position: relative; width: 100%; max-width: 900px; margin: 0 auto;"> <iframe src="https://shens2.github.io/smartart/" style="width: 100%; height: 500px; border: 1px solid #e0e0e0; border-radius: 8px;" frameborder="0" allowtransparency="true"> </iframe> </div>
